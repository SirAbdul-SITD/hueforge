Full description:
---------------------------------------------------------------
Hueforge is a color tool for anyone who works with color — designers,
developers, or anyone picking a palette.

Pick a color with hue/saturation/value sliders or type a hex code
directly. See it instantly in RGB, HSL, HSV, and hex, with one tap to
copy any format. Generate a 6-color harmony palette (complementary,
analogous, triadic) from any base color. And check real WCAG contrast
ratios between any two colors — the actual accessibility math, not a
cosmetic guess.

FEATURES
• Full HSV color picker with live hex/RGB/HSL/HSV conversion
• One-tap copy for any color format
• 6-color harmony palette generator (complementary, analogous, triadic)
• Real WCAG 2.x contrast ratio calculator with AA/AAA pass-fail for both
  normal and large text
• Save colors for quick reference later
• Fully offline. No ads, no accounts, no tracking.

Pick, convert, check contrast.
